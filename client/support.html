如何获取阿里云访问键ID和密钥？

请参阅 阿里官方文档。 简单来说，阿里云访问键主要用于OpenAPI调用，获取分为两步。
创建RAM用户并获取AccessKey。 赋予权限。本应用需要云服务器服务（AliyunECS）和专有网络（AliyunVPC）两部分权限。 只读用户赋予只读访问权限AliyunECSReadOnlyAccess和AliyunVPCReadOnlyAccess。 管理用户赋予管理权限AliyunECSFullAccess和AliyunVPCFullAccess。

如何写初始化脚本？

下面是个bash脚本（Ubuntu）的示例。建议以空行echo开始。更多示例请使用游客访问查看用户信息。

#!/bin/bash
# 启动约需1分钟
echo && echo "install user_data_pptp"

sudo apt-get update >/dev/null 2>&1
sudo DEBIAN_FRONTEND=noninteractive apt-get install -y pptpd >/dev/null

如何查询实例使用的费用？

请查询 阿里账单详情。 通过账单同时可查看实例的历史创建记录。

为什么实例详细页面正常显示，但是启动日志查看不了，初始化脚本的服务也连不上？

实例从资源准备到启动完毕大约需要１分钟时间。请在创建时间的１分钟后点击查看启动日志，并尝试连接初始化脚本中运行的服务。

为什么我的竞价实例在实例列表中没有显示？

如果你确定自己有竞价实例但是本应用没有显示，请在用户信息页面点击【退出登录】跳转到登录页面重新登录。登录时会刷新本地缓存。
